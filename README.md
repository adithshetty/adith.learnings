# adith.learnings
A log of everything I learn

## Getting Started

### Setup

1. Install dependencies:

```bash
npm i
```

2. Start the local development server:

```bash
npm run dev
```

3. Open the app in your browser at:

```text
http://localhost:3000
```

### Other Useful Commands

```bash
npm run build
npm run start
npm run lint
```
